# LaTeX / Math

Render formulas — mathematical expressions directly in the terminal.

## What LaTeX Rendering is

**LaTeX Rendering** converts mathematical formulas into Unicode symbols right in the terminal — a rare feature for a terminal viewer.

Einstein's famous equation: $E = mc^2$

Euler–Poisson integral:

```latex
\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
```
