# Tables

Structured data — rendered with Unicode borders and column alignment.

## What Table Rendering is

**Table Rendering** displays Markdown tables with box-drawing borders, column alignment, and support for inline code and LaTeX in cells.

| Feature | Syntax | Rendered | Align |
|:--------|:------:|----------|--------:|
| Code | `` `code` `` | `code` | `code` |
| Math | `$E=mc^2$` | $E=mc^2$ | $E=mc^2$ |
